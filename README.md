# Portfolio Website with Admin Panel

A modern portfolio website with an admin dashboard built using Next.js and Tailwind CSS.
It allows dynamic content management for sections like Home, About, Experience, Education, Projects, and Contact.

## Tech Stack

- **Next.js** – React framework for server-side rendering and routing
- **Tailwind CSS** – Utility-first CSS framework for fast UI development
- **JavaScript**
- **MongoDB**
